Sample source code for blog post explaining JS modules and module
loading

  Loading Javascript Modules
  http://libraryinstitute.wordpress.com/2010/12/01/loading-javascript-modules/

The commits should correspond to the steps in the blog post, with a few
dummy project files to illustrate the process of introducing modules and
module loading to a multi-script project. The payload is the newest
version of modules.js, but once you understand the basic ideas, you
might prefer one of the more feature-rich module loaders.
